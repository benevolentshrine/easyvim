# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in EasyVim:

1. **Do not** open a public issue
2. Email the maintainers or open a private security advisory
3. Include details about the vulnerability and steps to reproduce

We will respond within 48 hours and work to fix the issue promptly.

## Scope

EasyVim is a Neovim configuration. Security concerns may include:
- Malicious code in plugins
- Unsafe file operations
- Credential exposure

Thank you for helping keep EasyVim safe!
